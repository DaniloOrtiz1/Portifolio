# Portifolio
Este é o meu cartão de visitas no mundo do desenvolvimento. Aqui centralizo os projetos que desenvolvi (como a Serafim Perfumes, MG Joias e Ortiz Hair), demonstrando minhas habilidades em criar interfaces modernas, responsivas e focadas na experiência do usuário.

Neste projeto utilizei a "Trindade da Web" para garantir performance e compatibilidade total:

HTML5: Estruturação semântica e acessibilidade.

CSS3: Estilização com Flexbox e Grid para um layout totalmente responsivo.

JavaScript (Vanilla): Manipulação de DOM e lógica para navegação suave e filtragem de projetos.

Proximo Projeto: Em breve: Refatoração de toda a lógica para React.js e integração com Node.js para um blog de tecnologia.
